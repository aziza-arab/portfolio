SAE 1.02

Learn Conquer

Jeu ludo-pédagogique en Java

Interface du jeu Learn Conquer montrant la carte composée de tuiles et le déplacement du joueur.

Résumé : Le projet Learn Conquer est un jeu ludo-pédagogique développé en Java (iJava). Le joueur évolue sur une carte composée de tuiles représentant différents types de terrain. Certaines tuiles déclenchent des questions pédagogiques lorsque le joueur les atteint. L'objectif principal est de permettre à l'utilisateur d'apprendre en répondant à des questions tout en explorant une carte interactive.

Compétences techniques

  • Programmation en Java (iJava)
  • Gestion de structures de données (tableaux, cartes)
  • Lecture et utilisation de fichiers externes (.txt, .csv)
  • Implémentation d'une logique de jeu

Savoir-être

  • Travail en équipe
  • Rigueur dans le développement
  • Communication entre binôme
  • Organisation du travail

Livrables

  • Code source Java
  • Fichiers de données (.txt, .csv)
  • Classes Java (Question, Tuile, Matiere)

📊 Bilan : Cette SAE m'a permis de renforcer mes compétences en programmation Java et en structuration d'un projet de jeu. J'ai appris à organiser un code en plusieurs classes et à gérer des données externes via des fichiers.
